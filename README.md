Database Configuration (MANDATORY)To enable the "Community Predictions" sharing feature, you must configure the Firestore security rules in your linked Firebase project:Go to Firebase Console $\to$ Firestore Database $\to$ Rules.Paste and Publish the following rules to ensure public read/write access to the prediction history:rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /artifacts/{appId}/public/data/predictions/{documentId} {
      allow read: if request.auth != null;
      allow create: if request.auth != null;
    }
    match /artifacts/{appId}/users/{userId}/{document=**} {
      allow read, write: if false;
    }
  }
}
